# AiylBank

## Обзор проекта

AiylBank - это бэкэнд-сервис для управления банковскими счетами и транзакциями. Он предоставляет API для создания счетов, выполнения депозитов, переводов и получения выписок по счетам.

## Технологический стек

Проект разработан с использованием современных технологий для создания масштабируемых и надежных микросервисов:

*   **Java 17**: Основной язык программирования.
*   **Spring Boot**: Фреймворк для быстрой разработки автономных, готовых к производству приложений на Spring.
*   **Maven**: Инструмент для автоматизации сборки проектов.
*   **Spring Data JPA**: Для работы с базами данных и ORM.
*   **Lombok**: Для сокращения шаблонного кода.
*   **PostgreSQL (предполагается)**: Реляционная база данных для хранения информации о счетах и транзакциях.
*   **Docker**: Для контейнеризации приложения и его зависимостей.
*   **Docker Compose**: Для оркестрации многоконтейнерных Docker-приложений.
*   **JUnit 5 & Mockito**: Для написания модульных и интеграционных тестов.

## Начало работы

Следуйте этим инструкциям, чтобы запустить проект локально.

### Предварительные требования

Убедитесь, что у вас установлено следующее программное обеспечение:

*   **Java Development Kit (JDK) 17**
*   **Maven** (обычно поставляется с IDE или может быть установлен отдельно)
*   **Docker** и **Docker Compose**

### Запуск проекта с помощью Docker Compose

Рекомендуемый способ запуска проекта - использование Docker Compose, который автоматически поднимет базу данных и само приложение.

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/Sweezy352/Aiyl_Bank.git
    cd Aiyl_Bank
    ```
    
2. **Запустите Docker Compose**:
    ```bash
    docker compose up --build
    ```
    Эта команда соберет Docker-образы (если они еще не созданы) и запустит контейнеры для приложения и базы данных.

Приложение будет доступно по адресу `http://localhost:9091`.

### Запуск проекта локально (без Docker для приложения)

Если вы хотите запустить приложение локально, а базу данных в Docker:

1.  **Запустите только базу данных с Docker Compose**:
    ```bash
    docker compose up db
    ```
2.  **Соберите и запустите приложение с Maven**:
    ```bash
    ./mvnw spring-boot:run
    ```
    Убедитесь, что в вашем файле `application.yml` настроены правильные параметры подключения к базе данных, запущенной в Docker (например, `jdbc:postgresql://localhost:5438/aiyl_bank_database`).

## API Документация

Примеры запросов к API находятся в директории `http-client/`. Вы можете использовать их с любым HTTP-клиентом, поддерживающим формат `.http` (например, встроенный HTTP-клиент в IntelliJ IDEA).

### Swagger UI

После запуска приложения вы можете получить доступ к интерактивной документации API:

- **Swagger UI**: `http://localhost:9091/swagger-ui.html`
- **OpenAPI JSON**: `http://localhost:9091/api-docs`

Swagger UI предоставляет:
- Интерактивное тестирование всех эндпоинтов
- Подробное описание параметров запросов и ответов
- Примеры запросов и ответов
- Информацию о возможных кодах ошибок

## Тестирование

Проект включает Unit тесты. Вы можете запустить их с помощью Maven:

```bash
./mvnw test
```
